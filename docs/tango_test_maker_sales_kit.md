# 単語テストメーカー 販売キット

`web/tango-test-maker.html` を商品として販売するための一式です。
コピペしてそのまま使えるように書いてあります。

## 商品の要点(セールスポイント)

- 単語リストを貼り付けて2クリックで、印刷用の単語小テストが完成
- **A/B/C/D版**を自動生成 → 隣同士のカンニング防止
- 出題形式4種:英→日(記述)/ 日→英(記述)/ 英→日(4択)/ 日→英(4択)
- **採点用の解答一覧**も自動で付く
- Excel・スプレッドシートから2列コピーでそのまま貼り付けOK(区切り自動判定)
- インストール不要・完全オフライン動作・**生徒データが外部に一切送信されない**(学校の情報セキュリティ的に強い)
- HTMLファイル1個。ブラウザで開くだけ。買い切り。

## 法務メモ
