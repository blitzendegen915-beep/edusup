# 商品B:店舗ホームページ即日制作パッケージ 販売キット

`web/shop-site-template/index.html` を武器に、**1件 49,800円**のホームページ制作を売るためのキットです。

## なぜこれが「1時間で5万円」の最短ルートか

- 個人向け低単価商品で5万円 = 50件販売(非現実的)。**単価5万円×1件**に設計する
- ホームページ制作はココナラで5〜15万円が通る実績ジャンル。新規開業の飲食店・美容室・整体院は「インスタと食べログしかない」状態が普通で、需要が毎月再生産される
- 通常この価格が成立するのは制作に10時間以上かかるから。**このテンプレートは設定オブジェクト(CONFIG)を書き換えるだけで完成する**ので、受注後の実働は約1時間
- つまり「1時間で5万円」は誇張ではなく、**受注さえ取れれば実働1時間で5万円の粗利**という構造

## 納品ワークフロー(受注後・実働約1時間)

1. ヒアリングシート(下記)をLINE/メールで送って記入してもらう(お客様作業)
2. `index.html` を複製し、`PRESETS` の中から近い業種をベースに `CONFIG` を書き換える(30分)
   - 実際の納品ファイルでは `const CONFIG = PRESETS[...]` の行を `const CONFIG = {...お客様の内容...};` に置き換え、PRESETSは削除
3. お客様の写真を `heroPhoto` に設定、Googleマップ埋め込みURLを `mapEmbed` に設定(10分)
4. Netlify Drop(netlify.com/drop)にフォルダをドラッグ&ドロップ → 即公開(5分)
5. URLをお客様に送り、確認 → 修正 → 完了(15分)
