# 定期考査作成スキル 使い方ガイド

Claude Code のスラッシュコマンド `/exam` を使って、定期考査の問題を自動生成します。

## セットアップ

このリポジトリをクローンして Claude Code で開くだけで使えます。  
`.claude/commands/exam.md` がスキル本体です。

## 使い方

Claude Code のチャット欄で以下のように入力します：

```
/exam <科目名> <試験名>
```

### 例

| 入力 | 生成される試験 |
|------|--------------|
| `/exam 英語コミュニケーションII 2学期中間` | 2学年受験コース 英コミュII 2学期中間試験 |
| `/exam 英語コミュニケーションI 1学期期末` | 1学年 英コミュI 1学期期末試験 |
| `/exam 英単語テスト 第2回` | 1学年 第2回英単語試験（100問） |

## 追加指示の例

より具体的な試験を作りたい場合は、後ろに指示を追加できます：

```
/exam 英語コミュニケーションII 2学期中間 長文はSDGsをテーマにして
/exam 英単語テスト 第2回 教科書Unit3〜5の範囲で
/exam 英語コミュニケーションI 1学期期末 リスニングは会話文形式で2つ
```

## 出力内容

スキルを実行すると、以下が出力されます：

1. **問題本文**（大問ごとに番号・配点・指示文・問題）
2. **解答・解説**（各大問の正答と模範解答）

出力されたテキストを Word にコピー＆ペーストして体裁を整えてください。

## 科目別の特徴

### 英語コミュニケーションII（2学年受験コース）
- 大問6問構成、100点満点
- 空所補充・読解・語句選択・語句挿入・長文・並べ替え

### 英語コミュニケーションI（1学年）
- 大問8問構成、90点満点（Retelling含む）
- リスニング・語義選択・空所補充・並べ替え・アクセント・長文・英作文

### 英単語テスト（1学年）
- 全100問、100点満点
- マークシート（1〜80問）＋記述解答用紙（81〜100問）
- リスニング・英文定義・訳選択・日本語→英単語・文脈選択・記述

### 英語演習（スタイル学習中）
- 過去問がアップロードされ次第、スタイルガイドを更新します

## スタイルガイドの更新

新しい過去問が手に入ったら、以下をClaude Codeに伝えてください：

```
英語演習の過去問をアップロードしたので、/examスキルに学習させて
```

`.claude/commands/exam.md` の「英語演習」セクションが更新されます。

## 注意事項

- 問題の正解一意性・選択肢バランス・配点合計はClaude が自動チェックします
- 教材本文は明示的な指示なしに改変しません
- 個人情報（生徒名等）はダミーデータで実装します
- リスニング音声は別途用意が必要です（スクリプトのみ生成）
