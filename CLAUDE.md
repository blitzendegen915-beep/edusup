# CLAUDE.md

このリポジトリの方針・ルールは `AGENTS.md` に記載されています。必ず読んでから作業してください。

## 安全ルール

- `tasks/todo/` にあるタスク指示書を確認してから作業を始める
- `main` ブランチに直接コミットしない。必ず作業ブランチを切る
- 個人情報を含む可能性があるファイルは絶対に外部に送信しない
- 教材本文・テスト問題は指示なしに改変しない

## ブランチ運用

```
main           ← 安定版。直接触らない
└── claude/xxxx ← 作業ブランチ（タスクごとに作る）
```

