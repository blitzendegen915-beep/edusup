🛠 商品管理画面

1. GitHubに接続する

商品の保存には GitHub のアクセストークンが必要です（この端末のブラウザにのみ保存されます）。
作り方: GitHub → Settings → Developer settings → Fine-grained personal access tokens → Generate new token。 Repository access でこのリポジトリだけを選び、Permissions の Contents を Read and write にしてください。
