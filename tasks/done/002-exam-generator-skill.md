# タスク 002: 定期考査作成スキル

## ステータス: 完了（英語演習は保留）

## 概要

Claude Code のスラッシュコマンド `/exam` を作成。  
過去問のスタイルを学習し、新しい定期考査を生成する。

## 学習した過去問

| 科目 | ファイル | 学習内容 |
|------|--------|---------|
| 英語コミュニケーションII | PDF (5ページ) | 大問6構成・フォーマット・配点・問題形式 |
| 英語コミュニケーションI | PDF (冊子) + 解答用紙 | 大問8構成・リスニング・英作文・Retelling |
| 英単語テスト | 問題PDF + 解答用紙2種 | 全100問・マークシート+記述形式 |
| 英語演習 | **未アップロード** | スタイルガイド未登録 |

## 作成したファイル

- `.claude/commands/exam.md` — スキル本体
- `docs/exam_generator.md` — 使い方ガイド

## 使い方

```
/exam 英語コミュニケーションII 2学期中間
/exam 英語コミュニケーションI 1学期期末
/exam 英単語テスト 第2回
```
