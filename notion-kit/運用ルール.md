# Notion運用ルール（駒澤大学高等学校ラグビー部）

このドキュメントは `notion-kit` の各ページ・データベースを安全に運用するためのルールです。更新・追記は顧問（占部涼也）が行います。

参照元：`docs/rugby/基礎データ.md`（部員数・合宿情報等の実データはすべてこのファイルが正）。

## 2. 個人情報の扱い（公開ページ禁止事項）

📢ページ・連絡板データベースに書き込むときは、以下を**書かない**こと。

- 生徒のフルネーム（学年・パート・イニシャル程度の呼びかけに留める。例：「1年生の皆さんへ」「FW新入生対象」）
- 個人の体重・体脂肪率・食事内容などの記録
- 個人の試合成績・出場記録・怪我の状況
- 保護者の個人名・電話番号・住所（学校側の連絡先は顧問の署名のみ掲載可）
- 生徒同士を比較するような表現（「〇〇さんより良かった」等）

書いてよい内容の例：
- 「8/1（土）〜8/7（金）夏合宿（菅平ホテル）」のような日程・場所
- 「持ち物は雨具・タオル多めに」など全員向けの注意事項
- 「合宿費振込は◯月◯日締切」のような事務連絡（金額は家庭一律の金額のみ。個別の請求額などは載せない）
- 顧問（占部涼也／ryoyaurabe@gmail.com／080-5032-9150）の連絡先

迷ったら「これは学校の外に流出しても問題ないか」で判断し、少しでも不安があれば公開ページには書かず、LINE等の個別連絡や保護者会での直接説明に切り替える。

## 3. 食事記録の権限設計（なぜ生徒同士・家庭同士で見えないのか）

設計方針：**生徒・保護者はそもそもNotionのアカウントを持たない。**

- 生徒は「食事・体調記録フォーム」（Googleフォーム）に入力するだけで、Notionには一切アクセスしない。したがって「他の生徒のNotionページを見る」という経路自体が存在しない。
- フォームの回答は、生徒本人と回答を集計する顧問（フォーム管理者）だけが見られるGoogleスプレッドシートに集まる。生徒からは自分が送信した内容の確認（フォームの回答確認画面）はできるが、他人の回答は見られない。
- 集まったデータをNotionの「🍚食事コンディション記録」データベースに転記・取り込むのは顧問のみで、このデータベースはWeb公開せず、ポータル内の非公開ページに置く。
- 結果として、部員42名（選手38名・マネージャー4名。1年17名／2年12名／3年13名）の誰であっても、他の部員の食事・体調・体重の記録を見ることはできない。保護者同士も同様に、他の家庭の記録にアクセスする経路がない。
- 顧問が個別対応が必要と判断した場合（体調不良が続く、体重の急激な変化など）のみ、該当生徒・保護者に個別にLINE等で連絡する。記録そのものを他の生徒・保護者と共有することはない。

この設計により「Notionアカウントを登録させない」という前提を維持したまま、個人データを守ることができます。

## 4. 週次・月次の運用ルーチン（5分でできる更新作業）

### 毎週（5分）
- [ ] 今週の練習予定・変更があれば「📣連絡板」に1行追加（対象学年を指定）
- [ ] 直近1〜2週間に決まった試合があれば「📅試合カレンダー」に追加
- [ ] 顧問タスクボードの「進行中」タスクの状況を確認し、完了したものを「完了」へ移動

### 毎月（10〜15分）
- [ ] Googleフォーム「食事・体調記録フォーム」の回答スプレッドシートをCSVでエクスポートし、「🍚食事コンディション記録」DBに取り込む（または気になる生徒のみ手動転記）
- [ ] 体調評価が低い・体重変化が大きい生徒がいないか目視確認し、必要に応じて畠山監督と共有・対応
- [ ] 📢ページに古くなった連絡（終了した試合案内など）が残っていないか確認し、必要なら連絡板側で整理

### イベント前後（合宿・大会など。都度）
- 例：2026年夏合宿（8/1（土）〜8/7（金）、菅平ホテル）の場合
  - [ ] 日程確定後、速やかに「📅試合カレンダー」または「📣連絡板」に日程を掲載（金額など個別性の高い情報は載せず、日程・持ち物・集合時間など全員向け情報のみ）
  - [ ] 出発・解散時刻など変更が出た場合は「📣連絡板」で対象（全体／保護者向け）に追記
  - [ ] 合宿後は「✅顧問タスクボード」で会計報告書作成などの後続タスクを管理（非公開）

### 常時ルール
- 新しく連絡板・カレンダーに投稿する前に、本ドキュメント「2. 個人情報の扱い」を確認する。
- データベースのプロパティ（列）を追加・変更したときは、公開している📢ページ側のリンクドビューでも表示崩れがないか確認する。
- 顧問以外が更新作業を行う場合（後任の顧問など）は、本ドキュメント一式を必ず引き継ぐ。
