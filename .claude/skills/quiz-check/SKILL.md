# テスト問題の検品

作成・修正したすべての問題について、以下を1問ずつ確認してから納品する。

## チェックリスト

1. **正解の一意性**: 正解が1つに定まるか。別の選択肢も正解になり得る曖昧な問題がないか
2. **選択肢の偏り**: 正解の位置（A/B/C/D）が偏っていないか。選択肢の長さで正解が推測できないか
3. **配点**: 合計点が指定と一致するか。配点の記載漏れがないか
4. **設問番号**: 番号が連番になっているか。追加・削除でズレていないか
5. **英文の正確さ**: 文法・スペル・自然さ。出典教材の本文を勝手に改変していないか

## 問題を見つけた場合

修正してから再度このチェックリストを通す。修正内容は納品時の報告に含める。
