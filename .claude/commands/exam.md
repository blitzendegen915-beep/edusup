# 定期考査作成スキル

高校英語の定期考査を、過去問のスタイルに忠実に従って生成します。

## 使い方

```
/exam <科目名> <試験名> [追加指示]
```

**例:**
```
/exam 英語コミュニケーションII 2学期中間
/exam 英語コミュニケーションI 1学期期末
/exam 英単語テスト 第2回
/exam 英語演習 2学期中間
```

---

## あなたへの指示

ユーザーが `/exam` を呼び出したとき、以下の手順で定期考査を生成してください。

### Step 1: 引数を確認する

- `$ARGUMENTS` から「科目名」と「試験名」を読み取る
- 科目名が不明または未指定の場合は、ユーザーに確認する
- 学年・コース名が指定されていない場合は、過去問に合わせて推定する（例：英語コミュニケーションIIなら「2学年受験コース」）

### Step 2: 科目別スタイルガイドを参照して問題を生成する（下記参照）

### Step 3: 出力する

以下の順で出力する：

1. **表紙情報**（タイトル・実施日・受験上の注意）
2. **問題本文**（大問ごとに番号・配点・指示文・問題）
3. **解答・解説**（別セクションに、正答と簡単な根拠）

### 英語コミュニケーションII

**対象:** 2学年受験コース  
**満点:** 合計が100点になるよう配点を調整する  
**形式:** A4横書き、大問6問構成

#### 表紙

```
２０２６年２学年受験コース　英語コミュニケーションⅡ　○学期○○試験
```

- 全角文字・中央揃え・太字
- 試験期間に合わせて「中間試験」「期末試験」を変える

#### 大問構成（過去問に忠実に）

**大問１：空所補充（適語を入れる）〔9点〕**
- 日本文と英文のセット、9問
- 日本文に下線で重要語句を示す（例：に加えられる）
- 英文の空欄は `（　　　）` で表現。複数語の場合は空欄を複数並べる
- 空欄の前にアルファベットのヒントがある場合は `（ s　　　　）` のように示す
- 右端に解答のヒントとなる語句をハイライト（答案用の解答）
- 配点：1問1点（特定問題のみ2点）
- 問題例：
  ```
  (1) 彼の名前はそのリストに加えられるだろう．　　be added to
  His name will （　　　）（　　　）（　　　） the list.
  ```

**大問２：読解文＋設問〔6点〕**
- 対話文または説明文を枠線ありのボックスに入れる
- 設問1：「段落に適さない文を選べ」（①〜④から1つ）
- 設問2：空欄A・Bに入る接続詞などを語群から選ぶ（ア〜オの5択）
- 語群は `[ ア so / イ however / ウ similarly / エ what's more / オ in short ]` の形式

**大問３：語句選択（ア〜エから）〔4点〕**
- 選択肢4語を `ア ○○ イ ○○ ウ ○○ エ ○○` の形式で列挙（原形で提示）
- 本文中に `（　1　）` `（　2　）` のように番号付き空欄
- `All the options are given in their base form` の注記を英語で入れる

**大問４：語句の挿入位置特定〔10点〕**
- 5つの語句を枠に入れて示す（`0.are 1. is 2. what 3. across 4. been 5. the trouble`）
- 解答例を示す：`解答例　( cubes ) are ( widely )`
- 本文中の正解箇所には文中で色付き/太字で示さない（問題版）
- 解答欄：「語句が入る場所の前後の単語」を書く形式

**大問５：長文読解（内容一致4択）〔15点〕**
- 200〜300語程度の英文（段落は字下げ）
- 設問は5問、各3点
- `1. ○○ 2. ○○ 3. ○○ 4. ○○` の番号付き選択肢

**大問６：並べ替え（4番目・8番目）〔10点〕**
- 日本文 → 英文の語を並べ替える
- `（ ）` 内の4番目と8番目の語を答える
- 注意書き：「文頭に来る語も１文字目は小文字になっている」
- 5問、各2点

### 英語コミュニケーションI

**対象:** 1学年  
**満点:** 90点（Retelling含む）  
**形式:** 試験冊子形式、大問8問

#### 表紙

```
１学年
英語コミュニケーションⅠ
○学期○○試験
20XX. X. XX（曜日） 第X時限実施
```

受験上の注意を箇条書きで記載（リスニング放送は開始5分後の旨を含む）

#### 大問構成

**大問１：リスニング試験〔合計18点〕**

- **No.1**（1点×8 ＋ 9のみ2点）
  - 英文の空欄に聞き取った語を補う（①〜⑧の空欄）
  - ⑨ Question：直後の質問に英語で答える（解答欄59番）
  - 英文は２回放送

- **No.2**（1点×8 ＋ 18のみ2点）
  - 同形式・別の英文（⑩〜⑰の空欄）
  - ⑱ Question

- **No.3**（1点×5）
  - 対話を聞き、最後の文への応答を1〜3から選ぶ
  - 選択肢は全て放送される
  - 5問（解答欄19〜23）

**大問２：語義から単語選択〔2点×5＝10点〕**
- 英語で定義が与えられ、語群から記号（ア〜ク）で答える
- 語群は `ア. ○○ イ. ○○ …` の形式で8〜10語を列挙

**大問３：空所補充（日本語文→英語1語）〔2点×5＝10点〕**
- 日本語文 + 英文に1語の空欄
- `I （　　　）` のように英文の中に空欄

**大問４：並べ替え（番号で答える）〔2点×5＝10点〕**
- `[ 1. ○○ 2. ○○ … 6. ○○ ]` の語句群を並べ替え
- 解答は「（ ）の番号に該当する単語を数字で答える」
- 各問から2箇所の番号を答える（例: 34番目と35番目）
- 注意：「文頭に来るべき語も小文字にしてある」

**大問５：アクセント〔1点×4＝4点〕**
- 各語の最も強く読む箇所の記号（ア・イ・ウ・エ）を答える
- 単語を音節で区切って提示（例：`dy·nam·ic`）
- 4語提示

**大問６：長文読解（表の空所補充）〔2点×3＝6点〕**
- 200語程度の英文（短め）
- 人物についてまとめた表の空所を英語で埋める
- 解答は2語以上になる場合もあると注記

**大問７：長文読解（内容一致4択）〔2点×5＝10点〕**
- 250〜350語の英文
- 4段落構成を目安に
- 選択肢は `(a)〜(d)` の形式

**大問８：英作文〔3点×4＋3点＝15点〕**
- 問56〜59：日本語の意味を表す英文を書く
- 注意書きあり（指示がある場合はそれに従う）
- 問60：Retelling（指示に従い解答）
  - 採点基準：「文の形式を保っていない文・同一内容の繰り返し・質問内容を理解していないと思われるもの」を除き、3文以上→3点、2文→2点、1文→1点、0文→0点

### 英単語テスト

**対象:** 1学年  
**満点:** 100点  
**解答方法:** マークシート（1〜80番）＋記述解答用紙（81〜100番）  
**形式:** A4・全100問

#### 表紙

```
20XX 年度　X学年第X回英単語試験　　　　　20XX.X.X（曜日）実施
```

左揃え・太字、日付は右寄せで同じ行に配置

#### 大問構成

**【1】Listening〔10点〕**
- 「試験開始から約５分後に放送が始まります。」の注記
- 「放送される英単語の意味に当たるものを次の中から選び、記号をマークしなさい。英単語は２回ずつ流れます。」
- (1)〜(10)の10問、各4択（①②③④）
- 選択肢は `① ○○ ② ○○ ③ ○○ ④ ○○` の形式（横並び）
- 解答ページ末尾にリスニング問題の英単語一覧を記載

**【2】英文定義から単語を選択〔10点〕**
- (11)〜(15)の5問
- 英語で定義文が与えられる（例：`a feeling of wanting to know or learn about something`）
- 語群は①〜⑩の10択（各問で同じ語群）
- 語群は2行に分けて表示：
  ```
  ① ancestor  ② quality  ③ extend  ④ independent  ⑤ interest
  ⑥ model    ⑦ complete ⑧ similar ⑨ traffic     ⑩ vary
  ```
- 1問2点

**【3】英単語の訳を選択〔40点〕**
- (16)〜(35)の20問、各4択（①②③④）
- 形式：`(16) notice  ① を管理する ② と主張する ③ に気づく ④ を共有する`
- 1問2点

**【4】日本語→英単語を選択〔50点〕**
- (36)〜(60)の25問、各4択
- 形式：`(36) 正しい；適切な  ① rude ② common ③ social ④ correct`
- 複数の意味がある場合は「；」でつなぐ
- 1問2点

**【5】文脈空所補充（マークシート）〔20点〕**
- (61)〜(80)の20問
- 日本語文 → 英文の空欄に入る最適語を4択で選ぶ
- 形式：
  ```
  (61) 医師は医療機器を使って患者の胃を調べた。
  The doctor （　　） the patient's stomach with a medical device.
  ① controlled  ② determined  ③ approached  ④ examined
  ```
- 1問1点

**【6】記述空所補充〔20点〕**
- (81)〜(100)の20問
- 日本語文 → 英文の空欄に適語を書く（記述）
- ヒント：多くの問題で「与えられた文字から始めること（例：（ i　　　　）」
- 重要注意書き（太字・下線）：「**時制や主語、単複等による語形変化にも気をつけること。**」
- 1問1点
- 記述解答用紙（別紙）の3列グリッドに解答を書く

**末尾の注記:**
```
問題は以上です。スペルミス、マークミスや時制等、よく見直しましょう。
```

## 問題生成の共通ルール

1. **正解の一意性を必ず確認する**
   - 選択肢に正解が複数ないか確認
   - 空所補充で複数の正解がありえないか確認

2. **選択肢のバランスを確認する**
   - 正解が①に偏りすぎないよう、①〜④に均等に分散
   - ひっかけ選択肢は文法・語彙的に近い語を使う

3. **配点の合計を確認する**
   - 各大問の配点を明示し、合計が満点（英コミュII: 100点、英コミュI: 90点、英単語: 100点）になるか確認

4. **設問番号のズレがないか確認する**
   - 大問内の小問番号が連番になっているか
   - 解答欄番号と問題番号が一致しているか

5. **教材本文・テスト本文は勝手に改変しない**
   - ユーザーが提供した英文テキストをそのまま使う
   - 要約・改変が必要な場合は明示してから行う

6. **出力の最後に必ず解答欄を作る**
   - 大問ごとに正答を箇条書きで示す
   - 英作文は模範解答を示す

## 注意事項

- 問題本文中の英文はイタリック体で表示すべき部分（書名・固有名詞等）を `*斜体*` で示す
- 日本語の重要語句（キーワード）は `<下線>` または **太字** で示す
- 英単語テストのリスニング問題の英単語リストは、問題ページの末尾または別ページに記載
- 英語演習のスタイルガイドが登録されるまでは、英コミュIIに近い形式を参考に生成する
